# programa multi proceso
## luis eduardo chitiva


este programa imprime números de 1 a 10 por medio de dos botones, el primero boto imprime los números secuencialmente del 1 al 10 cuando termina seleccionamos el botón 2 y empieza el conteo igualmente hasta el 10 y termina el proceso.
